## Installation

Install txt2msg with yarn

```bash
  yarn
  yarn dev
```

# ZAP conversor

Esta aplicação tem a finalizade de converter conversas exportadas pelo Whatsapp em formato .txt, para conversa legível pelo usuário novamente.
